# ADs Engineering // Agency Website

A premium, high-performance web experience for **ADs Engineering**, a creative ad agency specialized in high-growth ecommerce brands.

## ✨ Features

- **Spline 3D Integration**: Interactive hero background with real-time 3D assets.
- **3D Testimonials**: Rotating perspective marquee for creator social proof.
- **Interactive Bottlenecks**: Sophisticated feature grid with dynamic hover states.
- **Stacked Sequence**: 3D-stacked workflow sequence cards.
- **Modern Stack**: Built with React, Tailwind CSS 4, and Framer Motion.
- **Micro-interactions**: High-fidelity transitions and glassmorphic UI.

## 🚀 Tech Stack

- **Framework**: Vite + React
- **Styling**: Tailwind CSS v4
- **Animation**: Motion (Framer), Spline Runtime
- **Deployment**: Optimized for high-velocity edge hosting
